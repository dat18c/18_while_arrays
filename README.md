## While loops
``````
int i=0;
while (i<10)
{
  System.out.println(i);
  i++;
}
``````
gør det samme som
```````
for (int i=0;i<10;i++)
{
  System.out.println(i);
}
```````

## Arrays

erklære

int tal[10]
``````

```````
mitTal = tal[7];
```````

```````
tal[7] = mitTal;
